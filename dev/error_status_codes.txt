-1 : SID login failure
    * 0 : wrong SID

-2 : email/password login failure
    * 0 : wrong email/password
    * 1 : different device login

-3 : sign up failure
    * 0 : invalid email format
    * 1 : email doesn't exist
    * 2 : email already registered
    * 3 : password too weak

-4 : team error
    * 0 : invalid coach email
    * 1 : invalid school
    * 2 : no team found to assign
    * 3 : database failure
